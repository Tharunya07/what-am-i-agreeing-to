# Proposed Labels

- `bug`: Confirmed defect or regression.
- `enhancement`: New feature or improvement request.
- `docs`: Documentation-only work.
- `chore`: Maintenance task (tooling, config, cleanup).
- `good first issue`: Small, well-scoped starter task.
- `blocked`: Cannot proceed due to dependency or external constraint.
- `priority:P0`: Critical; fix immediately.
- `priority:P1`: Important; schedule soon.
- `priority:P2`: Normal priority; planned backlog.
